# Security Policy

## Supported Versions
We are currently committed to supporting the latest stable version of this middleware.

| Version | Supported |
| :--- | :--- |
| 1.x | ✅ |

## Reporting a Vulnerability
We take the security of your applications seriously. If you have discovered a security vulnerability in this project, please report it privately before making it public.

**Please send your report to:** blacksanddesignsandbranding@gmail.com

### What to expect:
*   **Acknowledgment:** You will receive an initial response within 48 hours.
*   **Assessment:** We will verify the vulnerability and determine the impact.
*   **Resolution:** If confirmed, we will work on a patch and notify you before the disclosure is made public.

We appreciate your effort in helping us keep this middleware secure for everyone.
